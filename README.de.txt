Verwandle dein Shelly Wall Display in ein vollwertiges Smart-Home-Bedienpanel, das von Homey angetrieben wird. Die App betreibt einen lokalen Server auf deinem Homey, mit dem sich das Display verbindet -- so hast du Lichter, Steckdosen, Jalousien, Alarmanlagen und mehr direkt an der Wand im Griff.

Alle Geraete aus deinem Homey-System erscheinen als Kacheln auf dem Display, nach Raum gruppiert oder als flache Liste. Zustandsaenderungen werden in Echtzeit aktualisiert, ein Antippen schaltet das Geraet sofort. Das Dashboard unterstuetzt ausserdem Dimmer-Regler, Jalousie-Positionssteuerung, Sensorwerte (Temperatur, Luftfeuchtigkeit, CO2, Leistung), Kamera-Schnappschuesse und die freie Anordnung der Kacheln per Drag and Drop -- alles fuer den Touchbetrieb optimiert.

Das integrierte Energie-Dashboard zeigt ein animiertes Flussdiagramm mit den Verbindungen zwischen Solaranlage, Stromnetz, Zuhause und Batteriespeicher -- du siehst auf einen Blick, woher deine Energie kommt und wohin sie fliesst.

Weitere Funktionen:
- Dark Mode mit automatischer Speicherung der Einstellung
- PIN-Schutz fuer Alarmkacheln
- Unterstuetzung externer Sensoren (z.B. Reed-Kontakte fuer Garagentore), angezeigt als Offen / Geschlossen mit roter Kachel-Hervorhebung
- Einstellbare Kachelgroesse (XS bis XL) ueber die Einstellungsseite
- Auswahl, welche Geraete auf dem Dashboard erscheinen
- Energie-Dashboard kann unabhaengig aktiviert oder deaktiviert werden
- Die Dashboard-URL funktioniert auch in jedem Browser wie Chrome oder Safari -- nicht nur auf dem Wanddisplay
