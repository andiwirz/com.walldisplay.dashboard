Verwandle dein Shelly Wall Display in ein smartes Bedienpanel fuer dein Zuhause, angetrieben von Homey. Die App betreibt einen lokalen Server direkt auf deinem Homey und liefert ein touch-optimiertes Dashboard mit Echtzeit-Geraetesteuerung, Sensorwerten, einem animierten Energiefluss-Diagramm, Flow-Schnellstart-Buttons, Kamera-Schnappschuessen und PIN-geschuetzter Alarmverwaltung.

Geraete werden nach Raum geordnet oder als flache Liste angezeigt. Antippen zum Schalten, Schieberegler zum Dimmen, per Drag and Drop neu anordnen. Homey Flows lassen sich als Kacheln auf dem Dashboard einblenden und mit einem Tipp direkt ausfuehren. Dark Mode, einstellbare Kachelgroesse und ein integriertes Energie-Dashboard machen es zu einem vollwertigen Smart-Home-Wandpanel.
